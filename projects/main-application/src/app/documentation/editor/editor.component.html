Monaco Text Editor

This game uses the Monaco text editor. This is the editor that is used in the popular IDE VS Code. This editor is very powerful and contains all kinds of cool functionality out of the box. The editor is extended with functionality like syntax highlighting and auto-complete for the language used in this game.

You might or might not be familiar with Monaco or VS Code. To make sure you can get the most out of the editor and make the experience of writing code for this game as pleasant as possible, this page shows you some tips and tricks. Monaco has a huge amount of features, which are all available to help you code. It would be a lot of work to show all the functionality. This page only shows you the most important features you will most certainly use when playing this game.

Auto complete

Suggestions are automatically made while typing, and you can also press CTRL + SPACE for showing all possible suggestions. You can press TAB or ENTER keys to choose the suggestion. Some functions are filled in using a snippet that uses placeholders. After By pressing TAB you can go to the next placeholder. After the last placeholder, you can press TAB again to finish filling in the snippet. After finishing, you can see that the highlighting of the placeholders is removed.

Documentation on hover

Documentation about each of the keywords is included in the game. It is annoying to keep switching to the documentation page. So it is included in the editor! By hovering over a keyword with the mouse, you get the relevant documentation. Or you could place your cursor on the word and press CTRL + Q.

Note for people having experience with the Monaco editor: the default hotkey is CTRL + K CTRL + I (pressing two combinations after each other). This still works, but is not very practical. To make the documentation easier to access, the additional hotkey CTRL + Q is added.
