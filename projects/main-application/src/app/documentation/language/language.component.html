Aris

The language used in this game is called Aris.

Keywords

input: Represents the list that is considered the input of the level. It is only possible to retrieve values from the input, not add new values to it. When using the input (with for example move), you only have access to the first item. It is not possible to get any other item.

output: Represents the list is are considered the output of the level. It is only possible to add values to the output, not retrieve values from it. Values you put in the output (with for example move) must be as specified by the conditions of the level. If it is incorrect, the execution fails.

0, 1, 2, ...: Any digit represents a memory slot. Memory slots contain values, which can be further manipulated with other functions (for example incr). The number of available memory slots is defined per level.

Functions

Manipulating memory slots

move [memory-slot | input] to [memory-slot | output]: moves a value from one place to another.

copy [memory-slot] to [memory-slot | output]: copies a value from one place to another.

add [memory-slot] to [memory-slot]: adds the value of the first memory slot to the second. The first memory slot is not modified.

sub [memory-slot] from [memory-slot]: subtracts the value of the first memory slot from the second. The first memory slot is not modified.

incr [memory-slot]: increments the value of a memory slot by one.

decr [memory-slot]: decrease the value of a memory slot by one.

Loops

loop {{'{'}} [code block] {{'}'}}: Repeats the code block until the break command is used.

break: stops the execution of the inner-most loop. This method can not be used outside of loops.

continue: returns to the beginning of the code block of the loop. This method can not be used outside of loops.

If statements

ifzero [memory-block | input] {{'{'}} [code block] {{'}'}}: executes the code block if the value of the specified memory block or input equals 0.

ifnotzero [memory-block | input] {{'{'}} [code block] {{'}'}}: executes the code block if the value of the specified memory block or input is not equal to 0.

ifpos [memory-block | input] {{'{'}} [code block] {{'}'}}: executes the code block if the value of the specified memory block or input is positive.

ifneg [memory-block | input] {{'{'}} [code block] {{'}'}}: executes the code block if the value of the specified memory block or input is negative.
